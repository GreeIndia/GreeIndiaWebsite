# GREE INDIA - Frontend

## About Folder
The `frontend` folder contains the user-facing web application for GREE INDIA, built using React.js and Vite. It provides the interface for customers to view products, place orders, make service requests, and an admin dashboard for management.

## Folder Structure
- `src/`: Main source code
  - `components/`: Reusable React components
  - `pages/`: Page level components (e.g., CheckoutPage, AdminDashboard)
  - `config/`: Configuration files (e.g., API config)
  - `context/`: React Context providers for global state
  - `utils/`: Helper functions
  - `assets/`: Static assets like images and styles
- `public/`: Public assets

## How to Run
1. Navigate to the `frontend` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Build for production: `npm run build`
